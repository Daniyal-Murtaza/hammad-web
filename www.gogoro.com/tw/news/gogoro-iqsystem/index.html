為省用戶 15 秒花一年開發 Gogoro 如何把電動車秘密武器應用到二輪？

Gogoro 想太多

2015 年，Gogoro 做出世界第一台智慧電動機車；這個由 Gogoro 獨自開發的系統稱作 iQ System，包括 Gogoro 用戶熟悉的語音控制、動力調整、智慧感應解鎖、自動尋車、線上更新等功能，都靠它實現。

iQ System 就像是在車子上的 Android、Windows 等作業系統，再加上物聯網和大數據，寬廣的應用可能，正在顛覆車輛想像。

「是移動電腦、數據中心，或像手機裝上輪子」— 對於電動車加上軟體，許多投入電動車產業的企業領袖這樣形容；透過網路連線，電動車能遠端進行軟體配置與更新，用程式語言改安全性、操控性，調節動力，甚至添加人工智慧等各種功能。

有人把軟體視為電動車改變汽車產業的秘密武器，隨電動車企業取代傳統車廠成為全球最高市值的汽車公司，各車廠把軟體列為主力開發目標，宣告「軟體定義車輛 (Software Defines Vehicles )」時代來臨。

而 Gogoro 是世界首創「軟體定義」機車。

當其他電動機車還在討論動力和續航如何突破、當傳統燃油車還在摸索轉型，Gogoro 已能整合車體上的電腦、手機 App 和雲端網路，持續更新車子功能，領先把車輛產業的最新概念應用到機車並完成量產，究竟 Gogoro 是怎麼辦到的？

 

車子安裝大腦 iQ system「讓每個東西都能被控制」

Gogoro 換電網路和車子以智慧聞名，原因之一在於車子、換電站、電池都搭載了電腦晶片，透過網路和藍牙可連結雲端和個人手機，互享大數據並即時反應。

Gogoro iQ System 的原理，就是利用這些既有路徑開發， Gogoro 車子上的電腦 ECU (Electric Control Unit) 像大腦，透過線路、感應器等裝置掌握車子狀態，再以軟體程式對硬體下運作指令。

例如車廂開啟，用戶以手機 App 告訴 ECU 需求，ECU 透過軟體程式碼的運算，透過線路下達指令，控制車廂馬達何時該開啟或關閉。或像在尋車雷達功能裡，ECU接收手機 App 和車子的連線感應結果，經過線路控制，讓車子在特定時間自動發出聲音和燈光效果。

「我們儘可能做到，讓車子每個東西都可以被軟體控制」負責開發 iQ system的 Gogoro 產品企劃人員如此表示。不僅是上鎖與解鎖，車子的聲光、動力也都在 iQ system 一行行程式語言的掌控之下。由此不難想見，iQ system 從開發、測試到上市，花費多少心血，才能建構出系統的彈性與潛力，直到今天，仍不斷改良更新。

 

為省用戶 15 秒開發一年  重本打造創新體驗

有人說創新常伴隨一點瘋狂，這句話也在 Gogoro 對 iQ system 的研發上獲得體現。

Gogoro 為了讓 iQ system 的功能更加方便實用，持續投入大量的時間與心力。例如「自動感應解鎖功能」能讓 Gogoro 車子能自動偵測車主的靠近，並且只要一按 GO 鍵就解鎖，車主不用花時間掏鑰匙或卡片，只要輕輕一按就能直接發動。

這個功能的原理是，車子能自動偵測車主的手機藍牙訊號強弱，如果在短時間訊號由弱變強，iQ system 透過演算法就知道車主正在走過來，走到夠近時，車子就會自動做好解鎖的準備。

這個功能開發的困難之一在於，市面上的手機種類很多，不同型號的手機藍牙訊號也會略有不同，增加系統判斷車主距離的難度。於是為確保藍牙訊號的精確度， Gogoro 除不斷優化軟體演算法，也租用專業實驗室，模擬實際使用情境，用較常見的各型號手機測試。

「光是不同型號的手機就用了數百支，並且測試的精度是以公分為單位，讓手機從不同方向、角度靠近車子，反覆嘗試感應範圍，以此修正演算法判斷的條件，找出大多數手機都適用，並且車主使用起來最方便的方式。」Gogoro 產品研發人員說明開發過程。

「自動感應解鎖」上線後獲車主廣泛使用好評。這個功能為車主每次上下車節省約 15 秒掏鑰匙、收鑰匙的時間， 背後卻歷經了一年的研發時間。

iQ system 裡的各種功能都有類似的開發歷程，每項功能開發完成後，還需經長時間的上路實測，嘗試各種不同的使用情境，即便只是改動一個數值、聲音變化甚至小地方，都要確保不會影響其他功能運作，才會正式上線。

 

空中下載、雨天模式 聯網能力拓展應用可能

Gogoro iQ system 另一個打破傳統造車思維的功能，就是可以透過手機聯網更新車上系統。只要手機連上網路，並且以藍牙和車子連線，就可以利用 App 就可以更新車上的軟體版本、增加功能，不需要回廠也不用加裝硬體，就可以讓車子「愈騎愈新、愈騎愈聰明」。

Gogoro 這套線上更新的技術稱為空中韌體升級 FOTA（Firmware-Over-The-Air），身為世界首輛具備此技術的二輪載具，Gogoro 甚至開發出「斷點續傳」功能，如果藍牙連線中斷，當恢復時，韌體上傳的進度會從上次的資料斷點繼續開始，讓更新的速度更快、成功率更高。

而拜聯網能力之賜，iQ system 還可以接上網路世界、提取資訊，例如 Gogoro 的雨天智慧功能，就是利用手機 App ，經由網路獲得地點和天氣等資訊，若是下雨，App 會自動送出需求給車上的 ECU，切換成雨天模式，適度降低車子的起步扭力，避免濕滑路面造成打滑和危險，提升安全性。

能持續提升機車安全、功能和使用便利性，Gogoro iQ system 早早實現未來想像的關鍵，就是 Gogoro 兼具硬體和軟體的開發能力，造出適合的硬體作為行車基礎，搭配軟體、網路，有效開發聰明的功能，一個移動方式的典範轉移，正由 Gogoro 拉開序幕。
