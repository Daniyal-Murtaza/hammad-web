Gogoro 2020 旅遊攻略，跟著在地車友玩遍全台私房景點！

難得休假，想出去遊玩，卻又擔心開車會處處人擠人嗎？

旅遊就應該輕鬆愜意，不如現在開始給自己一個特別的旅遊計畫，來趟穿梭於街頭巷弄的 Gogoro 探索之旅。

我們為大家整理出在地車友們推薦的北、中、南、東台灣好去處，讓我們騎著 Gogoro 跟循車友們的足跡，一騎發現隱藏在細節裡的台灣之美吧！


北台灣：

全台灣人口密度最高的區域，在這個忙碌的城市，也許我們都只是匆匆走過，美好事物往往只能驚鴻一瞥、短暫停留。新的一年，我們不妨停下繁忙的腳步，載著你的愛人暢騎北台灣，細細品味北部獨有的美景風光吧！

基隆－美食美景都包覆著濃濃海味的基隆，在地人推薦的秘密打卡點－阿根納造船廠遺址。

台北－受盡了大都市的繁忙？推薦你來欣賞蘭陽大片的海景風光－盛夏光年景點，順道大啖－台北巷弄美食。

桃園－來場悠悠深度文化之旅，造訪融合環境之美的－晴耕雨讀小書院。

新竹－新竹不只有科學園區，還有暢通身心靈的百年神木－馬武督探索森林區。

陽明山為北部踏青旅遊首選，在一年四季有千變萬化的面貌，騎著 Gogoro 倘佯在其中，十足幸福。

晴耕雨讀小書院，周遭有大片草坪圍繞，無論白天黑夜，都是放鬆心靈好去處。

 

中台灣：

台灣的中心，富有薈萃的文風，同時也是文藝青年的聚集之地，處處皆有新鮮事，當你感到生活疲乏，需要激活，放假時刻不妨北下南上，一起到中部陶冶身心靈！

台中－傳說中在地人人「賞藝術、創藝術、藏藝術」好去處－大里台中軟體園區。

南投－群山環繞，如果你想遠離塵囂，推薦這片隱匿在山中的秘密仙境－忘憂森林。

南投群山環繞的絕美山景，不說我還以為來到了仙境。

Gogoro 無聲騎進山林，享受大自然的美好，忘憂森林，實如其名。

 

南台灣：

傳說中甜度最高，人情味最濃郁的地區，來這裡，你可以盡情豐收甜滋滋的笑容、吃不完的美食，與全台灣最暖的暖男暖女，當然還有不可或缺的大太陽！

嘉義－歷史在這裡留下味道。除了濃濃日本風情，還飄散著台灣檜木香－嘉義檜意森活村。

台南－溫暖重生，歷史的足跡，絲毫沒有離開過－陳世興古宅。

高雄－曾被歷史封塵的倉庫，因不斷注入創意靈感而重獲新生－駁二藝術特區。

屏東－萬巒不只有豬腳，還有全臺第一座極富創意趣味的 3D 彩繪吊橋－萬巒彩繪吊橋。

台南古色古香的陳世興古宅，騎著 Gogoro 無聲穿梭在巷弄裡，體驗古厝寧靜美好。

檜意生活村的一隅，處處都充滿巧思，處處都要與朋友來個歡笑三連拍！

 

東台灣：

囊括全台灣最驕傲的美景；最新鮮的空氣；最清澈的湖泊。不用我說，正港台灣人都知道東部是遠離塵囂、放鬆身心的好去處，讓車友們來教你怎麼玩！

花蓮－「實實在在的做人，正正當當的做冰」以這段話為精神的經營的冰店－正當冰。

宜蘭－坐擁兩塊猶如上天賜予的湖泊，一片是在由冬轉春之際，有機會看見絕美的落羽松的－蜊埤湖，另一片則是深藏在蓊鬱山林，周圍覆蓋著大片草地的－梅花湖。

花蓮正當冰店的鎮店之貓，身為貓奴不容錯過！

綠意盎然的梅花湖畔，隨著天氣變化景緻七十二變，無論春夏秋冬都要來張大合照！

花蓮伯朗大道，環島必訪景點，以連綿的山巒、翠綠的稻田、絕美的景緻為名！

海水蔚藍，遠眺壯闊山稜，山與海盡收眼底的花蓮七星潭。

 

Gogoro 新年快樂 2020 徽章出爐！

當我們騎著 Gogoro 無聲潛入各個景點，便能清楚感受到台灣這片土地、人文、美景，賜予給你最真實難忘的 2020 新年祝福。新年踩春去了嗎？不妨在 Facebook 及 Instagram 上 #LetsGogoro，和我們一騎分享你發現的台灣之美吧！
