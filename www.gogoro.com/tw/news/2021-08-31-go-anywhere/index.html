未來，換你出發！ Gogoro Network® 取得台灣中油 2021 電池交換站標案全數站點 網路佈建廣度深度齊備 熱區電池服務量能加倍 PBGN 車主一鼓作氣直上「武嶺」！

新聞稿

[ 8 月 31 日- 台北 ] 為滿足 Powered By Gogoro Network® (PBGN) 聯盟車款銷售快速成長的需求，Gogoro Network® 持續朝廣度、深度及密度三管齊下的策略佈建 GoStation® 電池交換站；截至目前為止，全台已部署 7,381 座 GoStation 電池交換機台，分佈在超商、超市、加油站、停車場等 2,143 個站點，提供超過 86,800 個智慧電池槽位供用戶交換。為持續增加服務量能，Gogoro Network 積極參與政府公建計畫（智慧電動機車能源補充設施普及計畫），並已順利取得台灣中油公司 110 年度電池交換站標案之全數站點（共 194 站），今年底完工後，勢必進一步提高可交換電池的供給量。不只站點的數量全面提升，Gogoro Network 也積極尋求各類型的合作夥伴，針對熱門旅遊景點佈建站點，全台逾 40 萬名 PBGN 車主今年底有望騎著智慧電動機車直達武嶺、暢遊日月潭、順騎花東縱谷，盡享不受限制的愜意旅程。

Gogoro Network 於新北市佈建的 GoStation 已覆蓋 96.6% 的行政區，為繼續加大服務深度與密度，再次與新北市政府攜手評估、規劃，期望於今年底前在新北市人口稠密的熱區部署近 50 站 GoStation。 Gogoro Network 目前在台北市、新北市、台中市和高雄市所佈建的 GoStation 站數，已超越當地加油站的數量；預計到今年底，桃園市和台南市的 GoStation 數量也有機會超越加油站，達成六都各城市的 GoStation 站數全面超越加油站的階段性目標。

Gogoro Network 總經理潘璟倫表示：「 GoStation 佈建策略是經由大數據解析台灣消費者的生活與移動型態，透過機器學習找出站點位置及規模的最佳推薦，讓數十萬用戶不需改變使用習慣，皆可以 24 小時全天候的享受更方便、更快速、更聰明的智慧電池交換服務。此次順利取得本年度台灣中油電池交換站標案，並與新北市政府密切合作，不但大幅提升電池交換站的覆蓋率和便利性，站點佈建廣度、深度及密度兼具，也讓騎乘 PBGN 電動機車進行都市通勤、長途旅行，甚至挑戰台灣公路最高點，都不再是問題。我們期待未來與更多的單位合作，更廣泛的滿足所有 Gogoro Network 用戶的需求。」

Gogoro Network 全新里程碑　未來，換你出發！
直達武嶺不是夢 日月潭環潭任你行 環島遊程更彈性

中橫公路與台灣公路海拔最高點「武嶺」，一直是廣大 Gogoro Network 用戶最想挑戰的旅遊路線和景點。目前位於南投埔里鎮的中油埔里中山路 GoStation 電池交換站，距離武嶺仍有 56.2 公里，垂直海拔爬升 2,862 公尺。今年的中油標案，有望於南投仁愛鄉聖本篤加油站增設一站 GoStation，屆時前往武嶺的騎行距離將縮短至 40 公里，垂直爬升高度降為 2,611 公尺。為推進前往武嶺的最後一哩路，Gogoro Network 與 7-ELEVEN 積極洽談，計畫在清境風景區內的見晴園門市架設另一站 GoStation，提供騎上武嶺最後 24.3 公里、垂直爬升 1,667 公尺所需的能源補給，預計在今年底前，PBGN 車主將能一鼓作氣、沿途換電池騎車征服武嶺。7-ELEVEN 是第一家與 Gogoro Network 合作的超商品牌，自 2018 年設立第一家門市至今已成功導入 190 家門市，助力 Gogoro Network 打造深入、融合生活圈的電池交換服務。

Gogoro Network 的電池交換站遍佈 301 個行政區，佔全台 368 個鄉鎮市行政區的 82%。過去幾年與台灣中油積極合作，全台座落於中油加油站的 GoStation 已達 437 站，中油也是架設最多電池交換站的單一合作夥伴。此次中油標案將讓 GoStation 首次進駐台東縣成功鎮、屏東縣竹田鄉、雲林縣四湖鄉、苗栗縣南庄鄉、頭屋鄉及南投仁愛鄉等六大鄉鎮，全面擴展電池交換站覆蓋廣度。PBGN 用戶未來不論要直登武嶺、日月潭環潭、規劃東、西部環島遊程，能源補充將更為便利，路線的選擇也可以更彈性、豐富多元。

永續能源應用　朝淨零碳排目標邁進
Gogoro Network 感謝各級政府一同拓展智慧綠能生活圈

新北市政府今年宣布「零碳翻轉、新北先行」，積極推動再生能源設置和運具電動化等措施。有鑑於此，Gogoro Network 攜手新北市政府，計畫於今年底再佈署近 50 站電池交換站，部分站點會設置於中和、永和等大量人口居住的熱區，有效提升車主電池交換的方便性。Gogoro Network 為持續擴大服務量能，上述站點都將以每站提供 100 顆以上智慧電池、可滿足超過 1,000 位用戶的電池交換需求的 Super GoStation 為優先考量建置方向。

Gogoro Network 感謝中央政府與地方政府持續打造電動機車友善使用環境，包括經濟部工業局與台灣中油透過公建計畫，擴大設置電池交換站作為能源補充設施；新北市政府透過公開招標，提供民眾更充足的電池交換站。相信在各界的共同努力下，台灣將朝淨零碳排的目標持續前進。
