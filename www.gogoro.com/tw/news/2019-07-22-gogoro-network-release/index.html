Gogoro 宣布 Gogoro Network 正式發展成為獨立事業體 全力聚焦已成市場主流的智慧電池交換服務 打造最多用戶與車廠選用的智慧交通工具開發平台

新聞稿

以打造智慧城市永續移動生態系為願景的領導品牌 Gogoro，在開放式的 Gogoro Network™ 智慧電池交換平台上，發展出廣受歡迎的 Smartscooter® 智慧雙輪，且帶動合作夥伴宏佳騰 (A Motor)、PGO、 YAMAHA 設計開發 Powered by Gogoro Network™ 電動機車，並於近期以新品牌 GoShare® 跨入共享機車領域。其中 Gogoro Network 提供的智慧電池交換服務，即將邁向 20 萬名用戶大關，更獲得許多海內外機車大廠的認同，成為台灣最穩定、也最多人使用的電動機車解決方案以及最多企業品牌選擇的電動機車商業模式。今日（22）Gogoro 正式宣布將 Gogoro Network 發展為全新的獨立事業體，專注在提供快速、簡單、聰明的開放式智慧電池交換平台以及智慧移動服務，加速將全世界各地的大型都會轉型成永續智慧城市。

Gogoro 執行長暨創辦人陸學森表示：「Gogoro 致力將大型都會轉型為具備永續移動生態系的智慧城市，點燃大眾市場邁向永續交通、智慧城市的渴望與熱情，讓所有消費者、在地社群、企業夥伴都可以成為打造更乾淨、更永續的城市運輸系統的重要一份子。在未來，Gogoro 將攜手更多的合作夥伴，在 Gogoro Network 這個開放式智慧電池交換平台上，提供更多產品與服務，讓消費者擁有更多元化的選擇，加速智慧移動產業的快速發展，實踐 Gogoro 永續智慧城市的願景。」

Powered by Gogoro Network™ 智慧電動車選擇多元
智慧電池交換模式帶動電動機車產業起飛

Gogoro Network™ 智慧電池交換平台，以其創新的電池交換為主，充電為輔的模式，翻轉了充電式電動機車充電時間漫長的現象。從 2015 年至今，全台灣購買電動機車的消費者中，有超過 95%的車主選擇以「電池交換」為基礎的智慧電動車。Gogoro Network 在其雙軌並行的模式下，僅有約 1% 左右的車主加購電池充電器，證明「電池交換」才是電動機車車主能源補充的主流方式，也是電動機車產業快速發展的重要方程式。截至目前為止，Gogoro Network 已累積突破 4,400 多萬的電池交換次數，平均每日交換次數更是高達 10 萬餘次，證明 Gogoro Network 在技術開發與用戶服務的堅持與承諾，打造一套最可靠、最值得信賴的智慧電池交換系統，成為台灣消費者轉換成電動機車時的首要選擇。

Powered by Gogoro Network™ 讓眾多車廠可以在其電控系統、智慧電池、電池交換平台等基礎之上，設計開發電動機車。目前，YAMAHA 已於日前正式推出 “ Powered by Gogoro Network™” 電動機車 EC-05，宏佳騰 (A Motor) 與 PGO 也預計於下半年接著發表全新電動車款，將帶給台灣消費者更多元的選擇。購買 Powered by Gogoro Network™ 智慧電動車的消費者，不論選擇的是哪一個品牌及車款，皆可享受 Gogoro Network 所提供的快速、簡單、聰明的智慧電池交換平台服務。

GoStation® 電池交換站遍佈全台 數量與密度皆全台第一
AI 管理系統以及智慧電池不斷升級

Gogoro Network 總經理潘璟倫表示：「Gogoro Network™ 為目前世界最大的智慧電池交換平台，接下來除了持續專注於智慧電池技術的提升並透過 AI 人工智慧優化電池交換服務與 GoStation® 電池交換站的佈局外，也將全力支援更多合作夥伴開發 Powered by Gogoro Network™ 智慧電動車，同時更將提供夥伴開發更多以此開放式平台和電池技術為基礎的創新服務型態，持續帶給消費者更多元化的選擇與更好的體驗。」

Gogoro Network™ 主要涵蓋四個關鍵要素：

  1. 雲端基礎設施：Gogoro Network 整合了 AI 人工智慧、機器學習等強大技術，創造了新一代的智慧電池交換平台。搭配 24 小時 Network Operation Center (“NoC”) 雲端網路管理中心，即時提供便利、高效率的服務，是全世界最穩定、最具規模的電池交換系統。
  2. GoStation® 電池交換站：透過完整的管理系統，使用者可至城市中隨處可見的GoStation® 電池交換站點快速的交換智慧電池。Gogoro Network™ 電池交換站數量全台第一，全台佈局密度最高，是業界首見讓車主在 24 小時內完成環島的電動機車解決方案。目前 Gogoro Network 在全台灣的電池交換站，包含即將啟用的站點已突破 1,292 站，合作夥伴包括中油、台塑等直營或加盟加油站超過 390 站，7-ELEVEN、全家、全聯、萊爾富等便利商店等超過 400 站，桃園市政府停車場、台北捷運等政府單位超過 100 站。到 2019 年底前，Gogoro Network 預計將在全台建置達 1,500 站的 GoStation® 電池交換站，在人口密集的都會區，車主平均騎車 3 分鐘，就會有一座電池交換站。
  3. Gogoro Network™ 智慧電池：智慧電池結合了多項突破性的專利技術，在安全性與效能方面均領先業界。此外，智慧電池具備完善的自我防護機制，可持續更新、優化和執行電池健康診斷。2019 年 7 月底起，Gogoro Network 也將逐步開始投放採用 21700 電池芯的 Gogoro Network™ 智慧電池到市場中，車主們將有機會交換到性能提升達 27%並且比競品多出 2.8 倍容量 (Ah)的電池，騎乘里程數將可達 170 公里，是市面上同級車款相同測試條件下，表現最佳的組合。
  4. Gogoro Network™ App：正式推出獨創的 Gogoro Network™ App，引領使用者迅速且輕鬆的找到最近的 GoStation® 電池交換站，並且隨時進行電池服務資費方案變更、疑難排除與帳務管理，更能查詢歷史騎乘紀錄。此一站式的解決方案可以確保 Gogoro Network™ 使用者獲得最多與最有價值的資訊。

另外，Gogoro Network 也積極跨界合作，提供使用者更多元、更優惠的加值服務。近期即聯手遠傳電信推出史上首見的「雙飽」方案。搭配遠傳門號月付 $398 起，可享受行動上網吃到飽，同時獨享 Gogoro Network™ $699 電池服務資費騎到飽的限期優惠；並擴大兆豐 Gogoro 聯名卡現行方案，所有購買 Powered by Gogoro Network™ 電動機車的車主也可於即日起同享每月 5% 電池服務資費現金回饋優惠。Gogoro 相信在這樣全新的營運規劃之下，對於消費者與合作夥伴而言，將能獲得更多更便利的能源服務與資源，欲瞭解更多 Gogoro Network™品牌資訊請參考官方網站：https://network.gogoro.com/tw/。
