Gogoro Network® 用戶突破 54 萬，自由省新資費方案 6/1 上線！新購車限時加碼資費優惠，每月多享近 530 公里免費騎

新聞稿

[ 5 月 31 日- 台北 ] Gogoro 旗下電池交換平台 Gogoro Network 突破 54 萬電動機車用戶數，於全台佈建逾 2,500 個 GoStation 站點、12,000 個機櫃1、已投放超過 110 萬顆智慧電池，每日支援逾 40 萬次電池交換次數。持續滿足多元騎乘需求，Gogoro Network 於 6/1 起，新增 兩個全新「自由省 $519」及「自由省 $819」電池服務資費方案2，讓自由省資費級距更完整，搭配動態折價3、遞延折扣4 與免費性能提升服務5，騎多少、花多少，都超值。同時，Gogoro Network 祭出新購車限時優惠，6/1 起至 12/31 入手 Gogoro 或 PBGN（Powered by Gogoro Network）之二顆電池（含以上）任一款全新智慧電動機車，如當月選擇「高用量自由省 $969」或「高用量自由省 $1,269」電池服務資費方案6，即加贈電池使用費折抵金 $1,000 元，相當於每月多享近 530 公里7 免費騎！

多元自由省方案

騎多少、怎麼省，由你決定！

以今年第一季來看，已有超過五成 Gogoro Network 新用戶訂閱「自由省」電池資費方案8，依據實際電池使用量計費，怎麼騎、怎麼省，都由自己掌握。自 6/1 起，Gogoro Network 將在現行自由省 $319 方案的基礎上，新增「自由省 $519」及「自由省 $819」方案（適用於二顆電池），為車主提供更彈性靈活的選項，騎乘更輕鬆自在。
