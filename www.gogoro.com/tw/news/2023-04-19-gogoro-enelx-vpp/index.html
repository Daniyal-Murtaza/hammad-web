Gogoro 與 Enel X 義電智慧能源 共同加速台灣能源轉型，Enel X 虛擬電廠整合全台逾 1,000 個電池交換站點、2,500 座機台， 宣布全球首個整合虛擬電廠與電池交換能源網路的商業部署

新聞稿

台北， 2023 年 4 月 19 日 – 致力於打造都會永續移動解決方案的全球電池交換生態系技術領導者 Gogoro®（納斯達克股票代碼：GGR），與創新能源服務及虛擬電廠龍頭　Enel X （義電智慧能源），今（19）日重磅宣佈部署全台逾 1,000 個 Gogoro Network 電池交換站點、​2,500 座機台加入 Enel X 虛擬電廠（Virtual Power Plant, VPP），參與台電電力交易平台的電力輔助服務市場，實現全球首個整合虛擬電廠與電池交換能源網路的商業部署，推動台灣再生能源轉型。此次大規模部署源於 2022 年兩間公司成功的前導測試計畫，亦為全球第一個以此技術商轉成功的案例。

Gogoro 創辦人暨執行長陸學森表示：「我們正邁入智慧能源基礎建設的新時代，透過整合 Gogoro Network 與 Enel X 虛擬電廠，為台灣市場帶來創新的電力資源，這更是此項技術全球首度落地商轉，並為我們帶來移動產業以外的新營收來源。我們相信電能時間移轉（time shifting energy）是實現能源與交通永續轉型的關鍵，Gogoro 更持續扮演驅動這場創新變革的重要角色，每一位 Gogoro 用戶也都是台灣淨零轉型的貢獻者。依據台灣的電網需求，Gogoro 將協助整合可用電能回饋至電網中。當電網的用電負載增加，我們的電池交換站可隨時接受調度，暫時停止充電、或將智慧電池中儲存的電力饋電至電網中，同時不影響車主的換電使用，達到更即時、有效的能源平衡管理。」

Gogoro與 Enel X 的虛擬電廠於 2022 年展開前導測試計畫，驗證了當電網失衡時，Gogoro Network 電池交換站能夠安全降載、或在必要時饋電至電網。一旦電網恢復平衡， Enel X 的虛擬電廠會向 Gogoro Network 發送指令，使其電池交換站恢復一般用電。Gogoro Network 電池交換站在完成降載與饋電的同時，仍可讓消費者安心使用既有的電池交換服務，突顯 Gogoro Network 電池交換系統是支持台灣能源轉型的理想電力資源。聚合了 Gogoro Network 電池交換站的 Enel X 虛擬電廠，其功效與電力業者新建電廠以滿足用電需求相當。

雙方已於本月部署 500 個 Gogoro Network 電池交換站點、近 1,300 座電池交換機台參與台電電力交易平台輔助服務市場，並計劃於今年年中擴增至共 1,000 個站點、2,500 座機台，支援電網平衡與穩定運作。

台電總經理王耀庭表示：「Gogoro 與 Enel X（義電智慧能源）的合作由先前的前導測試計畫擴大到實現商業化落地，並將拓展至逾 1,000 個 Gogoro Network 電池交換站點，是台電輔助服務提高電力使用效率的重要里程碑。這也是全球電力產業大量使用再生能源的發展過程中必要的一項重要工作，更為企業與分散式資源積極投入台灣電力市場樹立新典範。 Enel X 的虛擬電廠聚合 Gogoro Network 電池交換站的電能負載，參與台電電力交易平台的輔助服務。我們的電力交易平台，允許像 Gogoro 一樣的企業成為電力產銷者，協助推動城市穩定電網平衡與運作。我們期許更多企業透過像 Enel X 的電力聚合商加入，共同為台灣能源轉型貢獻力量。」 

Enel X 亞太區總裁 Jeff Renaud 表示：「我們與 Gogoro 的合作驗證了虛擬電廠是再生能源轉型的重要推手。新的分散式電力資源加速能源使用的電力化和去碳化，隨著這些新型電力資源的建立，虛擬電廠能集結這些資源並使其連接到更廣大的能源系統，進一步發揮更大的永續影響力，同時增加營收。我們的平台將成千上萬個『新電力資源』整合為單一資源，藉其協助平衡大型再生能源發電的間歇性，維持電力穩定。我們很高興此次攜手 Gogoro 樹立全球領先的虛擬電廠應用新典範，也期待未來持續擴大合作。」

據台電推估，未來電力共享經濟商機將上看新台幣百億元規模1。Enel X 在台灣能源市場上擁有最大的虛擬電廠，在台電電力交易平台中管理最多分散式電力資源2。

關於 Gogoro Network® 智慧電池交換網路 : 

Gogoro 提出有別於傳統能源補給的嶄新解決方案 Gogoro Network — 奠基於電池交換與智慧移動服務的開放平台。Gogoro Network 集資訊連結、人工智慧、機器學習於一體，從而建立新一代電池交換與能源補充方式，它能夠聰明且可擴展地根據使用者、不同族群及商業目的，做動態與多元的連續自我優化。Powered by Gogoro Network (PBGN) 方案則提供 Gogoro 車廠夥伴，運用 Gogoro 創新及智慧解決方案 (不論是智慧動力總成、控制器、元件或是智慧系統) 的商業模式，各個車廠都可以發展與推出整合使用 Gogoro Network 與交換電池的獨特電動車輛。

關於 Enel X（義電智慧能源）

Enel X（義電智慧能源）是義電集團（Enel Group）下專注於先進能源服務的事業線。作為發展創新解決方案以支持能源轉型的全球領航者，Enel X 提供消費者、企業與城市依客戶需求建構的模組化整合方案，並以推動電力化及數位化作為創造嶄新價值的驅動力。Enel X 管理的需量反應服務，全球容量合計約有 8.5 GW，並已裝置 75 MW的表後儲能，且在全球裝設超過3百萬盞公共照明。此外，每日提供約 6,700 萬個住宅用戶能源服務。Enel X 的解決方案生態系統包括能源優化及自產能源的資產、能源效率的優質解決方案，以及具競爭力且彈性的能源產品，並以幫助客戶發展自己的能源規劃為終極目標。
