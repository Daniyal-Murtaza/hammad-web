Gogoro 今年缺席米蘭機車展，原來去了這裏！

連續兩年米蘭車展 (EICMA) 最受矚目的 Gogoro，今年居然沒有現身米蘭這個可以說是 Gogoro 進軍歐洲的應許之地！同一時間，Gogoro 的智慧雙輪和電池交換站，依然飄洋過海，出現在同屬南歐的葡萄牙里斯本，參加地表最強科技盛會之一的 Web Summit，與 Google, Uber, Booking.com, BMW, Mercedes Benz, IBM, Volkswagen, AWS, TOYOTA, LEGO 等科技龍頭、汽車品牌和新創公司一起與會。

 

沒聽過 Web Summit，別說你是科技控！

有別於大家熟悉的拉斯維加斯 CES、柏林 IFA、巴賽隆納 MWC、台北 Computex，Web Summit 是近年來知名度快速竄升的全球性科技與新創盛事，不論是參與的公司、活動的安排、演講的主題、參觀的人士，都充滿新創科技公司的活力以及科技控的熱情。

今年 Gogoro 首次參展，與 Google 為鄰，展出 Gogoro 的智慧雙輪與能源網路系統。Horace 也親自飛往里斯本，與超過 2000 名的新創、科技界 CEO 交流，並參加兩場座談會，分別是 11 月 9 日與巴黎市長、巴賽隆納的科技長一起討論智慧城市將如何創造新世代的智慧生活，以及 11 月 7 日和 My Taxi 的 CEO、歐洲議會的交通與觀光主席探討車輛共乘下的共享經濟。


11/7 Tue.  20:00（台北時間）  
車輛共乘下的共享經濟 / Pooling our carpooling networks: Ridesharing as a global gamechanger 

11/9 Thu.  21:20（台北時間）
智慧城市將如何創造新世代的智慧生活 / Smart cities for smart living

  

Web Summit 是最新的車展

今年最重要的主題是 Auto / Tech，擁有百年技術的汽機車公司們絕對是今年 Web Summit 最潮的科技新貴。BMW 的智慧電動車 i8 就放在入口最顯眼的地方，不但可以預約試駕，更將由 BMW產品開發高階主管分享汽車介面數位、電子化的趨勢。Volkswagen 的 CIO 將介紹超級電腦在未來將如何駕馭馬路與交通。低碳運輸、無人駕駛、共享經濟都是 2017 Web Summit 的焦點，當車廠跳脫傳統思維模式的同時，Gogoro 正與全世界最聰明的思想家與發明家，一起在新科技的浪頭上重新定義交通與移動。

 

神秘嘉賓霍金探討 AI 未來發展

知名物理學家史蒂芬霍金透過視訊，現身 Web Summit 的開幕活動，預警人工智慧可能對人類造成的影響，帶來首波高潮。霍金指出，人工智慧可能是發生在人類史上最好也是最糟的事情，全世界的科學家們應以謀求人類的福祉為初衷，來發展人工智慧。更多精彩的開幕活動內容，請前往 Web Summit 臉書粉絲團觀看直播影片。



未來三天，Gogoro 團隊將從里斯本現場實況轉播，帶回第一手的科技新知！各位關注智慧交通、科技創新的粉絲們，千萬別錯過了。
