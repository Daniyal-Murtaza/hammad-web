防疫安心 Go Gogoro® 貼心挺你 首推二輪業界最優「我 OK 你先騎，0 元入手」防疫購車專案 期滿最高 51% 保證買回 加碼首年資費優惠

新聞稿

[ 6 月 15 日- 台北 ] 國內新冠疫情嚴峻，全台三級警戒持續延長，原本仰賴大眾運輸系統的民眾及有移動需求的消費者，對個人移動交通工具的需求日益提升。面對疫情，Gogoro 為減輕消費者購車負擔，特別推出「我 OK 你先騎，0 元入手1」防疫購車專案，針對指定車款民眾可以選擇「超低月付分期付款＋前三個月無需繳款＋3 年後保證最高 51%買回2」的嶄新購車方案，同時再加碼首年資費折扣優惠，希望幫助消費者在疫情期間，也能輕鬆無負擔的入手 Gogoro 智慧電動機車，完成個人移動的生活需求。

三個月後再付 即刻 0 元入手
三年保證買回 最高 51% 超保值

為減輕消費者在疫情期間的財務負擔與提供更靈活的財務規劃，6/16 起至 8/1 止，凡購買 Gogoro 1、Gogoro 2、Gogoro 3 和 S Performance 系列車款，都可選擇「 0 元入手 三年保證買回」貸款專案，可享有 ０ 元交車、三個月後再支付最低每日 $42元3 的超低日付金額，讓消費者的資金運用更加彈性。前 36 個月車主只需負擔 53% 車價和分期付款利息，Gogoro 在三年期滿後提供四種配套方案供車主選擇，包括：1. 續辦分期擁有愛車、2. 一次結清擁有愛車、3. 保證買回、4. 保證買回並換購新車（最高 51% ），提供疫情期間更多元的購車方案，與消費者一起共體時艱。

指定車款再享一年電池資費折扣 $2,400 元

除了「 0 元入手 三年保證買回」專案，Gogoro 還加碼推出指定車款再享至少一年電池資費折扣 $2,400 元的優惠4，購買包括 2021 / 20 年式的 Gogoro S2 / S3、Gogoro 2 / 3 Plus、Gogoro 2 / 3 GT、Gogoro 2 / 3 Delight 、Gogoro 2 Rumbler 等車款，都可享有 12 個月或以上，每月折扣「電池服務資費 200 元」的優惠。疫情期間，不僅能夠輕鬆入主 Gogoro 智慧電動機車，還能搭配優惠資費方案和政府購車補助，減輕購車壓力，讓民眾可以放心購車、安心騎車，「我OK 你先騎，0元入手」防疫購車專案詳細內容請參考活動網頁：https://promotion.gogoro.com/tw/summer-campaign-sp/

客製化直播賞車火熱 短短兩周突破兩萬人次觀看流量

Gogoro 自六月初推出客製化直播賞車服務以來，整體已超過兩萬人次觀看，開創新型態賞車方式。此外，Gogoro 官方網站除提供完整的線上 3D 賞車體驗外，消費者可依據喜好的車款及車色，自由地增添搭配各式個性化配件，打造出心目中理想的智慧電動機車，並提供完整的政府補助、相關貸款及分期付款方式之試算。線上需求送出後，將有專人聯繫消費者，讓客戶透過遠距賞車也能即時又深入的鑑賞心儀車款。

Gogoro 與你一起攜手抗疫 度過難關

因應疫情發展，Gogoro Network® 對換電站除了定期巡檢清潔外，並搭配額外滾動式增加消毒次數，尤其強化觸控螢幕與電池握把部分，確保每一位用戶都能安心換電；同時也提醒用戶務必經常清潔並消毒車輛的龍頭、手把等手部容易接觸殘留病毒的部分，才能安心享受綠能騎行。

Gogoro 全台服務門市與維修中心，同步實施高規格的防疫措施，除了配合政府法令採行實聯制、量測體溫和控管室內人流外，店內展示、試乘、待交車輛每日定時消毒；同時所有門市同仁全日皆配戴口罩、定時用酒精消毒手部與傳送每日健康管理回報。各地維修中心亦同步升級防疫措施，針對有車輛維修保養需求的車主，Gogoro 啟動室外接車，室內一組、室外一組的人流管制標準，確實執行收車後的進廠車輛消毒，交車前的出廠車輛消毒，除了保障消費者能安心騎乘，也保護門市夥伴的健康。

備註：

1. 0 元入手係指本專案購車可使用專案分期付款，且融資公司不收取分期付款設定費等其他費用，惟專案參加人仍須支付其他交車費用，包含但不限於：文件處理費、領牌代辦費等及需另外簽署 Gogoro Network 智慧電池合約使用電池服務。

2. 相關限制適用，詳活動注意事項以及參加人與融資公司間契約規定。

保值最高 51%：本專案保證買回金額，為參加人參加本專案時所購入車輛車價（「專案車價」）之 47%。但如主張保證買回時，另向 Gogoro 購入另一台新車，則可獲得相當於專案車價 4% 之購車金，用以折抵參加人購買新車之車價。

3. 最低每日 $42元 是以 New Gogoro 3 專案分期期間每月月付金額 $1,279 計算，實際金額以融資公司約定準。

4. 一年電池資費折扣係指指定車款首 12 期帳單週期 Gogoro Network 電池服務資費每期折抵 $200 元，依消費者購買車款之不同，可能適用不同資費折扣優惠，詳細規則見 Gogoro 網站說明。
