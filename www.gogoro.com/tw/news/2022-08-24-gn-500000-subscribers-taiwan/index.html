感謝有你 +1 Gogoro Network 用戶突破 50 萬 價值 100 萬資費折抵大方送 快加入電動機車行列 電動機車市場上半年銷售逆勢成長 電池交換車種深受民眾青睞

新聞稿

[ 8 月 24 日 - 台北 ] Gogoro 旗下電池交換平台 Gogoro Network®  今 (24) 日宣布已突破 50 萬電動機車用戶數，寫下台灣智慧能源發展史的重要里程碑。為感謝 50 萬車主支持，Gogoro 將在今年下半年推出一系列慶祝活動及全新計畫，並持續投資、擴大網路服務規模。首先登場的是 Gogoro Network「感謝有你 +1！50 萬騎士達成，新舊朋友都有禮、百萬獎勵送給你」的慶祝活動，即日起至 9/25，Gogoro 或 PBGN 車主完成指定任務，或是任一消費者新購任一款 Powered by Gogoro Network 智慧電動機車，都有機會獲得連續 25~50 個月的電池資費折抵，獎勵價值總額高達新台幣 100 萬元。歡迎第 50 萬用戶加入，Gogoro 也將於 9/17 在桃園中壢大潤發，舉行年度車主活動，揪大家一起放電享受夏日能量派對，活動詳情將於近期公布在官網。

全球運具電動化的時代加速來臨，今年台灣機車市場因疫情和原物料大漲等因素影響，今年 1~7 月整體機車市場與去年同期相比下滑 1.2%，燃油機車銷售更下滑 2.9%；反觀電動機車的銷售數量則逆勢成長高達 13.4%1，民眾購車選擇更向電動機車挪移，共同迎接電動機車黃金年代。

今年上半年，全台灣採用電池交換的電動機車，已佔全體電動機車銷售量的 96.1%2，消費者對電池交換能源補充方式的接受度遠勝於自行充電；而在所有領牌的電池交換電動機車中，高達 95.9% 的用戶皆選擇 Powered by Gogoro Network (PBGN) 聯盟車款3，突顯出 PBGN 聯盟各品牌所推出的超過 40 款車種，深受消費者喜愛與信賴。

各品牌努力推行運具電動化的成果逐年展現，Gogoro Network 的用戶數在 2021 年 7 月達成 40 萬的里程碑後，於短短 14 個月後又迅速增加 10 萬，達到 50 萬用戶的重要里程碑。2015 到 2022 的 7 年間，Gogoro Network 從 0 開始，一路累積至 50 萬用戶，期間不斷的建設與改良，GoStation 已遍布全台，數量與密度不只是全台第一，也是市場上最受歡迎、且最值得信賴的電池交換系統，更是全世界最大規模的電池交換網路。

除了衷心感謝消費者的支持與眾多合作夥伴不遺餘力推廣電池交換服務外，Gogoro Network 承諾將持續投資，全力打造更廣泛的用戶生活圈，相信隨著更多 Super GoStation 的佈建與啟用，用戶將會享有更便利的電池交換體驗。

全台 Super GoStation 建置超過 130 站

服務總量與覆蓋率皆穩定成長

伴隨著用戶數快速增長，Gogoro Network 已在全台灣佈建超過 10,918 個電池交換機櫃，可提供服務的電池交換槽總數達到 98,720 個，分布在涵蓋全台各城市的 2,240 個站點；單一站點具備 120 個或以上電池交換槽的 Super GoStation 已建置多達 133 座，僅 Super GoStation 所提供的電池交換槽數就高達 16,210 個，隨時可以提供大量、高效、智慧與多工的電池交換和能源服務。

Gogoro Network 透過增設 Super GoStation 與 GoStation，不僅擴大單一站點的服務能量，也讓整體網路佈建更深入大眾生活圈，在便利商店、量販店、加油站、停車場、捷運站和餐廳等日常移動場景，都會發現 GoStation 全年、全時、全天候的提供穩定服務。Gogoro Network 致力讓每一位車主騎乘更便利、安心與享受。除了延續推動 GoStation 佈建，更與各級政府單位合作建置郊區、偏鄉的電池交換站點。東台灣第一座 Super GoStation「台東好站」也已於八月初正式啟用，提供台東當地與環島車主更多服務量能，未來我們將持續優化網路、提高客戶滿意度，並發展多元智慧電池運用方案，更好的響應循環經濟訴求。

百萬獎勵送給你 達成任務可享電池資費折抵

「來電 50，集綠點抽資費折抵」任務  邀請 Gogoro Network 用戶探索「綠點」交換電池

「50 萬再 +1，每月輕鬆騎」活動  邀請民眾加入 PBGN 行列

為感謝 50 萬用戶的信賴與支持，達成全體用戶超越 3.2 億顆電池交換的里程碑，Gogoro Network 特別舉辦「來電 50，集綠點抽資費折抵」活動，邀請用戶於 8/24 14:00 ~ 9/25 23:59 間至全台「綠點」GoStation 交換電池，截圖該綠點站點可供交換滿格電池數字，上傳一張或加總數張截圖數字大於或等於「50」時，即可參加「每月資費現折 200 元 x 連續 25 個月」抽獎，活動總計將抽出 100 位幸運 PBGN 車主。

迎接電動機車黃金年代，Gogoro Network 也邀請更多消費者一起加入 PBGN 交換電池的行列，與上述活動同步推出「50 萬再 +1，每月輕鬆騎」活動，活動期間凡購買任一款 PBGN 智慧電動機車且付清車款，完成購車後登錄活動表單，即可參與購車「每月資費現折 200 元 x 連續 50 個月」的抽獎活動，活動將於 9/27 抽出 50 位幸運新車主，贈送總額高達 50 萬的資費折抵。

欲知以上活動詳情請瀏覽 Gogoro Network 官網：https://network.gogoro.com/tw/promotions/500k-celebration/?state=welcome50
