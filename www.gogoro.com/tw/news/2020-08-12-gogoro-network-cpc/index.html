Gogoro Network™ 參與台灣中油公開標案再傳捷報 2020年新增 110站　遍佈19個縣市 大幅提升「非都會區 」GoStation® 覆蓋率

新聞稿

[ 8 月 12 日- 台北 ] 為滿足 Powered by Gogoro Network™ 電動機車聯盟銷售的快速成長，Gogoro Network 持續快速佈建電池交換站。截至目前為止，全台累計有1,7931 站電池交換站，在六大都會區，用戶平均騎車 3 分鐘2就可以享受到快速便捷的換電服務。為了持續增加非都會區的覆蓋率，Gogoro Network 積極參與台灣中油的公開標案，順利獲得「109 年台灣中油公司加油站場地電動機車電池交換站設備使用競標方案」中的 110 站點，密切與台灣中油攜手合作，加速電池交換站的覆蓋密度。在2020 年底，Gogoro Network 預計有超過 1,900 座 GoStation® 電池交換站，覆蓋全台 298 鄉鎮市行政區，其中，在台灣中油的站點將可達 348 多站，是電池交換站數量最多的單一合作夥伴，足見對綠能電動運具發展的重視。

Gogoro Network™ 總經理潘璟倫表示：「隨著用戶們的騎乘路線更多元，加油站點一直以來都是能源補充的起點，更是生活通勤的驛站。感謝台灣中油公司長期以來推廣電池交換服務不遺餘力，台灣中油是 GoStation® 第一個合作夥伴，從2015 年 5 月台灣中油永錡泰山加油站啟用以來，Gogoro Network一直持續緊密與台灣中油合作，此次標案得標站點更是大幅提升 GoStation® 在「非都會區」的覆蓋率，為其區域用戶提供更便利的服務。展望未來，Gogoro Network也將持續與推廣電池交換服務的重要夥伴們合作，打造更方便的電池交換生活圈。」

Gogoro Network 再次插旗離島 進軍小琉球

繼 2018 年 Gogoro Network 於澎湖建置首座離島的電池交換站後，小琉球將成為 Gogoro Network 進軍外島的第二個據點。看準小琉球是旅遊觀光的熱門地區，機車在小琉球更是主要的移動交通工具，透過綠色運具廣泛使用，在開心觀光之餘，還能減少空污及對環境的破壞。除了離島建置外，Gogoro Network 更補足西部和東部的旅遊路線，拓展 14 個先前尚未建置的行政區，從西部的苗栗大湖鄉、公館鄉、後龍鎮、獅潭鄉、通霄鎮、雲林褒忠鄉、嘉義東石鄉、中埔鄉、鹿草鄉和台南七股區，南至屏東琉球鄉和新埤鄉；東部則是台東東河鄉和鹿野鄉，讓騎乘旅遊更加輕鬆便利。

不僅大幅提升 GoStation® 電池交換站在全台鄉鎮市的覆蓋率，佈建範圍更深入大眾生活，在便利商店、量販店、加油站、停車場等日常生活圈所見之場所，皆可看見電池交換站的蹤影，讓換電不只更加便利且快速。

Gogoro 自創立以來始終將環境議題視為品牌責任之一，不斷地透過實際行動，倡議永續發展的重要性。Gogoro 零排放的智慧雙輪、PP 材質的車殼，都是在努力減少對環境的傷害；其永續經營、注重環境發展的理念，更是與「RE-THINK 重新思考」不謀而合。日前由Gogoro 與 「RE-THINK 重新思考」聯手舉辦夏日環島淨灘活動於 26 日圓滿落幕，為期九天五場的淨灘活動，超過 2 千人的參與、清除 7,825 公斤海廢垃圾、守護 39 萬平方公尺海灘面積，約達 56 座足球場，創下「最廣淨灘總面積」全新驚人紀錄，加上長期關注環境議題的「台客劇場」參與，也帶動更多民眾一同共襄盛舉為環境盡一份力。
