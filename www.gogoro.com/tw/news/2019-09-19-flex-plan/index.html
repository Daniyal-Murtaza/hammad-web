自由騎 聰明省 Gogoro Network™ 推出創新「自由省」資費方案 智慧動態折價機制，讓用戶騎得更輕鬆自在

新聞稿

Gogoro Network 同時宣佈推出全新設計的 GoStation® 3.0 電池交換站、Super GoStation 超級電池交換站，並大量投放新一代的智慧電池

[ 9 月 19 日- 台北 ] Gogoro Network 為全球最大的智慧電池交換平台，專注於提供快速、簡單、聰明的開放式智慧電池交換服務。今日發表創新的「自由省」電池服務資費方案：不僅依照大數據即時智慧分析，透過動態折價平衡網路供需，更免費贈送「性能提升服務」，讓所有用戶都享有更彈性、聰明的額外選擇。

此外，Gogoro Network 同步推出全新的 GoStation® 3.0 電池交換站，外觀設計簡潔，與前代 GoStation® 電池交換站相比，更為緊湊的機櫃卻能儲放更高數量的電池，且充電效率更高。另，Gogoro Network 也開始大量投放新一代的智慧電池，並規劃在最繁忙的熱點區域建置 Super GoStation 超級電池交換站，以利用戶在尖峰時段亦能享有高品質的智慧電池交換服務。

Gogoro 執行長暨創辦人陸學森表示：「Gogoro Network 是領先業界的開放式智慧電池交換平台，以創新的 Swap and Go 六秒快速電池交換體驗，解決都會人口稠密區電動車輛能源補給的惱人問題，讓一般用戶與商業用戶都能聰明、簡單、輕易採用。Gogoro Network 導入結合 AI 人工智慧與即時大數據分析的 SmartGEN 2 技術，推出革命性的『自由省』電池資費服務方案，其智慧動態折價機制，讓用戶騎得更輕鬆自在，怎麼騎，怎麼省，都能自主決定。」

含動態折價優惠的「自由省」電池服務資費方案
「自由省」方案是專為 Gogoro Network 用戶設計的全新的電池服務資費方案，依據實際電池使用量計費。此資費方案一大亮點為動態折價優惠，引導用戶至非尖峰時段或鄰近使用量較低的 GoStation® 電池交換站點，透過改變用戶的電池交換時間或地點，Gogoro Network 提供最高 20% 的動態折價，回饋每位用戶對平衡電網需求所作出的貢獻。

「自由省」電池服務資費方案預計於10月份先行推出 BETA 版本，所有的 Gogoro Network 用戶月繳 $299 元即可認購，方案用戶每月可獲贈至高等值於新台幣200元的電池使用費折抵金，並以每安培小時 (Ampere Hour) $2.3 元計算實際電池使用費，享有無限次數電池交換、可輕易在 Gogoro® App 或是 Gogoro Network™ App 找到動態折價站點資訊、免費獲贈價值每月 $249 元的性能提升方案等優惠。此外，年底前搶先認購者可再獲贈連續3個月每月 $100 元的額外折抵金。

Super GoStation® 超級電池交換站
Gogoro Network 持續優化能源網路效率，提供用戶最佳體驗，已於今年4月開始建置可容納超過 100 顆電池的 Super GoStation® 超級電池交換站，透過人工智慧與先進的演算法，進行優化，讓用戶隨時可享有更聰明的電池交換服務。每一座超級電池交換站可儲存與管理超過 200 kWh 容量的電池，能夠同時滿足超過 1,000 名車主的電池交換需求。除了目前已經開始營運的 3 座超級電池交換站之外，Gogoro Network 計畫在 2019 年底前，於六都各佈建一座以上的超級電池交換站。

全新 GoStation® 3.0 電池交換站
Gogoro Network 推出一款全新 GoStation® 3.0 電池交換站的設計，較前一代 GoStation® 增加高達 50％ 的電池承載量，同時大幅減少 GoStation® 佈建所需要的站點整體面積及空間。隨著容納電池數量的增加，若電力中斷時，全新 GoStation® 3.0 電池交換站可自主供電高達 64 小時，繼續在不穩定的供電環境中提供最穩定的能源補充服務。GoStation® 3.0 並具有提供雙向充放電、快速充電、微電網 (Micro Grid) 電力調配等的前瞻設計。

Gogoro Network™ 新一代智慧電池
Gogoro Network 於4月發表基於 21700 電池芯技術研發的新一代智慧電池，不僅單顆電池蓄電容量較前代產品提升了 27%，達到業界最高 1,740Wh，因為 Gogoro Network 與時俱進的堅持，讓過往販售的任一輛智慧雙輪都能與新一代電池完全兼容，所有用戶無需做任何改變，即刻享有新科技帶來的便利、效能和騎乘距離。新電池已於8月份開始投入 Gogoro Netowrk 中營運，陸續將有更大量的新一代電池加入網路服務用戶。
