2017 新裝登場，5 件你不得不知的新鮮事

一直以來，將「創新」視為企業核心之一的 Gogoro ，於即日 ( 08 / 18 ) 起新裝登場，要讓更多消費者見識 Gogoro 的嶄新魅力！以下是 5 件最重要的新鮮事：

自在操駕

前輪 SPORT 性能胎 ，騎車更得心應手

Gogoro 為了滿足車主對於操控性的期望與要求，於 2017 年的全系列 Smartscooter® 智慧雙輪前輪上使用高性能的 SPORT 性能胎，以取代原本的雙能複合胎。這款與  Maxxis 合作開發的性能胎，一樣採用等間距 Gogoro® 專屬胎紋設計，具有良好的排水性，並搭配賽車用的胎面膠料（半熱熔胎），可以發揮優越抓地性。換上 SPORT 性能胎後的 Smartscooter®智慧雙輪，可以有效縮短煞車距離，彎道表現也更加穩定順暢。

全新質感

粉體塗裝部件，質感提昇而且更加環保

2017 全系列的 Gogoro 在前避震搖臂及多連桿後避震等部件，皆以粉體塗裝取代液體塗裝，其低調霧面的質感與觸感，不但著色均勻，更可幫助減少磨損與褪色，是 Gogoro 對於設計細節的追求。此外，粉體塗裝可以大幅減少塗料的浪費，並且不使用具揮發性的有機溶劑，對於作業人員、環境和空氣品質的影響降到最低，是目前較先進的塗裝方式，廣為汽車工業及高級自行車等產品使用。

耐用升級

全封閉式皮帶護蓋，延長皮帶使用壽命，減少耗材使用量

有鑑於較差路況的地面或使用環境有較多的碎石或沙塵時，彈起的砂石容易造成傳動皮帶的提早磨損，因此 2017 全系列車款皆改用全封閉式皮帶護蓋，有助於隔絕髒污與塵土，使傳動皮帶工作在清潔的環境中運作。封閉式護蓋沒有經過拆除或外力破壞等情形，且都仍須按時回廠進行定期檢查與保養的狀況下耐用里程也倍增為二萬公里，建議更換里程可達兩萬公里，讓所有 Gogoro 騎士的旅程更無後顧之憂。

方便省力

全新主腳架設計，輕鬆駐車更省力

回應車主在駐車上的需求與建議，Gogoro 將全面採用新款主腳架。改良的槓桿幾何設計讓駐車更加省力，也能幫助避免車子被不當移動時的意外傾倒。車主停車更輕鬆也更放心！

貼心設計

行李箱小燈有定時熄滅功能，不再擔憂意外耗電

回應車主建議，Gogoro 的行李箱小燈將增加定時熄滅功能，不用再擔憂因為忘記關閉行李箱導致電池耗電殆盡，無法發動，同時，此配件將改為選配，消費者可根據自身需求選購。

最後，相較於 2015 年款， 2017 年新裝也做了些微改變，以更貼近使用者實際需求，以下說明：

  • 2015 年款座墊的鎖扣為雙環式，雖然牢靠，但曾有車主反應不易對位，因此 2017 年款改成符合市場通行的單環式。
  • 全系列的置物箱裡設置橡膠蓋，讓維修檢查更快捷，節省您寶貴的時間。
  • 行李箱小燈及 USB 插孔取消，改為選配。
  • 煞車油量檢視為服務中心執行定期保養時的 SOP 之一，因此 Gogoro Plus 取消煞車油檢視燈。
  • 安全帽掛勾將改為沉穩內斂的黑色。
  • Gogoro Lite 將取消前後定位燈，但頭燈的近燈會全時點亮，跟定位燈的作用一樣，安全無虞。
  • Gogoro 版本後扶手上的磁扣座改為選配。
  • 除網路商店之外，取消車前銘板。透過網路商店購買仍可自訂刻字。
  • 為幫助延長傳動皮帶建議更換里程，傳動皮帶護蓋將改為全封閉式皮帶護蓋， Gogoro 及 Gogoro Plus 因此取消音頻導輪，車輛可能會在特定時速時產生皮帶共振產生的聲音，但此屬正常現象，倘於一般室外環境，此共振聲音亦不易察覺，也不會影響任何行車安全，請您放心。

歡迎您即刻至門市賞車，搶先擁有 2017 年新裝，為您的旅程加分！
