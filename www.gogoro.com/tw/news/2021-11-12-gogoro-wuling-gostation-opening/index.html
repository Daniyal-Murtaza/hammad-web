Gogoro Network® 正式開通中橫沿線站點 沿途部署百顆電池助 PBGN 電動機車直上台灣公路最高點「武嶺」 同場加映阿里山公路站點於 11 月底前啟動營運 從富貴角到鵝鑾鼻 從合歡山到阿里山 壯遊全台暢行無阻

新聞稿

 [ 11 月 12 日 - 南投 ] Gogoro Network 今 (12) 日宣布，由埔里前往合歡山武嶺旅遊區沿線 GoStation® 電池交換站正式上線，Powered by Gogoro Network (PBGN) 車主不僅可以騎乘電動機車暢意環台，還可深入有台灣屋脊之稱的中央山脈，探索獨特的高山美景。持續增加服務量能，滿足廣大騎士們的多元需求，Gogoro Network 同時宣布開通前往「嘉義阿里山風景區」與「南投日月潭風景區」主要省道旁的多個電池交換站，並於 11 月底前陸續啟動營運。從富貴角到鵝鑾鼻、從合歡山到阿里山，Gogoro Network 積極拓展多種國民遊憩踏青路線，讓全台逾 43 萬名 PBGN 車主可盡情享受零碳排的山水美景之旅。

要蓋就蓋大的　要玩就玩真的

壯遊挑戰合歡山　遠征探索阿里山　恣意暢遊日月潭

從埔里直奔合歡山武嶺旅遊路線，Gogoro Network 已完成清境「 7-ELEVEN 見晴園店站」、省道台 14 號上「中油台一加油站」和「中油聖本篤加油站」電池交換站的佈建並即刻啟用，供應 100 顆電池給欲登高的所有 PBGN 車主。位於清境風景區內 7-ELEVEN 見晴園店旁的 GoStation，可提供超過 20 位用戶一起探索武嶺，滿足騎上武嶺最後 24.3 公里、垂直爬升 1,667 公尺、達到海拔 3,275 公尺所需的能源補給。電池交換站開通首日，即有 22 位車主熱血直衝合歡山，包含於 2015 年購買 Gogoro 1 的第一位車主、Gogoro 首席試車手及多位資深車主，還有宏佳騰智慧電車、PGO 電動車、台鈴 eReady 與 Yamaha 等 PBGN 夥伴的車主，騎乘 20 多輛各式 Gogoro 與 PBGN 電動機車、以及 GoShare 共享機車車款直上武嶺，輕鬆寫意壯遊合歡山。

不僅如此，Gogoro Network 亦持續加強、完善經由不同路線前往「日月潭風景區」的電池交換站：包括從埔里往南出發沿省道台 21 線，新增了中油巨大站、中油魚池站兩個 GoStation 站點，提供超過 50 顆的電池讓騎士暢快奔馳；或是從南投市及名間鄉向東出發，除了在集集鎮原有的兩個 GoStation 站點外，於省道台 16 線上新增了中油新集集站與中油集鹿站，搭配現有位於日月潭邊的台灣聯通日月潭中興場站及水里鄉的兩個站點，日月潭景區周邊有眾多的電池交換站提供車主選擇，輕鬆進行無碳環保親水之旅！

今日 Gogoro Network 也加碼宣布，預計 11 月底前完成阿里山公路（省道台 18 線）直達阿里山風景區的電池交換站佈建，沿路新設立的 GoStation 站點包括：中油信泰站、中油吳鳳廟站及中油石卓站，在阿里山公路沿線投放超過 80 顆電池，讓全台各地的 PBGN 車主能夠以省道台 1 線、台 9 線搭配台 18 線，隨時啟程近身體驗無與倫比的阿里山美景。

PBGN 車主周遊全台灣 想 GO 就 GO 行程不受限

2017 年 Gogoro Network 首次串起西部走廊，電動機車實現北高縱走；2019 年初，能源網絡終於全台串聯，樹立史無前例的「全程零碳排電動機車環島」創舉。根據 Gogoro Network 的大數據分析，迄今為止共有 4,655 位車主在過去兩年半內，完成超過 6,381 趟環島旅程。環島解鎖是 Gogoro Network 發展的必經過程，除了持續提升西部都會區的能源網路密度，也不斷地在台灣東部的公路上佈建更多的電池交換站，讓每一個 PBGN 車主都能順騎花東縱谷，說換就換、想停就停、說走就走。

Gogoro Network 目前營運及建置中的電池交換機櫃已超過 10,723 個，廣布在全台 2,332 個位置，是全球最大、最穩定、最聰明且最多人使用的電池交換系統。Gogoro Network 佈建深入大眾生活圈，於便利商店、量販店、加油站、停車場、捷運站、餐廳以及風景區等日常移動範圍皆可見其蹤影。如今用戶數已突破 43 萬，累計超過 2.2 億顆的電池交換。除了透過增設電池交換機櫃，提高對用戶的服務廣度；Gogoro Network 也將繼續積極投資興建可提供 100 顆以上智慧電池、滿足超過 1,000 位用戶電池交換需求的 Super GoStation，使電池交換網路的深度與廣度同步發展，滿足大量成長電動機車車主隨時、隨地的能源補充需求。

統計今年一月至今，Gogoro Network 已在全台建置了超過 1,579 個電池交換機櫃，平均每天增設 5.2 個機櫃（每個交換機櫃有 8~10 個電池槽位）。同時，Gogoro Network 為確保用電安全與充電效率，每個電池交換站均依據其服務量體，單獨向台電公司申請對應電力。一方面以充裕的電能、提供車友最穩定的換電服務，另一方面遵循電工法規配置電路，並依站點座落位置進行換電動線規劃，在全台灣打造最安全、最穩定、最便利的電動機車電池交換網路。
